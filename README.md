# Onion Trace — Website

A single-page marketing site for **Onion Trace**, a blockchain-intelligence platform
that traces cryptocurrency transactions to their true end receiver.

## Run it

Just open `index.html` in any browser. No build step, no dependencies.

## Add team photos

Drop square (1:1) images into `assets/team/` with these exact names:

| Member  | File path                  |
|---------|----------------------------|
| Vinay   | `assets/team/vinay.jpg`    |
| Shriraj | `assets/team/shriraj.jpg`  |
| Rahul   | `assets/team/rahul.jpg`    |
| Sairam  | `assets/team/sairam.jpg`   |

Until a photo is added, each card shows a styled initial as a placeholder —
the photo replaces it automatically once the file exists.

## Files

- `index.html` — page content & structure
- `styles.css` — all styling (dark theme, responsive)
- `script.js` — nav, scroll reveals, counters, animated hero network
